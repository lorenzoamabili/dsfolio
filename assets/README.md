# Assets Folder

Place your images and files here:

## Required Images

- `avatar.png` — Your profile photo (160×160 px, square)
- `favicon.ico` — Browser tab icon (16×16 or 32×32 px)
- `project-1.jpg` — Thumbnail for project one (420×280 px recommended)
- `project-2.jpg` — Thumbnail for project two (420×280 px recommended)
- `project-3.jpg` — Thumbnail for project three (420×280 px recommended)

## Tips

- Use high-quality images (at least 2x the display size for retina screens)
- Keep file sizes reasonable (< 500KB per image)
- Use JPG for photos, PNG for graphics with transparency
- You can generate placeholder images at https://placehold.co/ if needed

Example placeholder URLs (replace with real images later):
- Project thumbnails: `https://placehold.co/420x280/00BFA6/FFFFFF/png?text=Project+1`
- Avatar: `https://placehold.co/160x160/00BFA6/FFFFFF/png?text=You`
